Archive of Soviet Trumpet Music

About this project

A community-driven archive collecting scores, recordings, and metadata for Soviet-era trumpet music. This project aims to document composers, pieces, and historical context to make them discoverable and preservable.

What is included

  • Composer metadata (names, nationalities, dates)
  • Composition titles and basic bibliographic data
  • Links to public domain recordings and scores where available

Contributing

If you have materials, corrections, or references to add, please use the "Make additions to the archive" link on the main page. Contributions should include a source or reference where possible. If you prefer, open an issue on the project's repository or email the maintainers with details.

Technical notes

This site is built with Astro. The composers page fetches rows from a spreadsheet-backed server API; the client UI groups rows into composer cards and provides search filters.
